# Day 29 - Nodemon & Postman

## What is Nodemon?
Nodemon is a utility tool for Node.js that automatically restarts the server whenever file changes are detected.

## What is Postman?
Postman is a popular API testing tool used to send requests to a server and check responses without creating a frontend application.

## Installation

### Install Dependencies
npm install express
npm install --save-dev nodemon

### Run Project
npm run dev

## Nodemon Workflow
1. Write Code
2. Save File
3. Nodemon Detects Change
4. Server Restarts Automatically
5. Updated Output

## Technologies Used
- Node.js
- Express.js
- Nodemon
- Postman
